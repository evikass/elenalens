# ElenaLens — сайт-визитка фотографа

Премиум одностраничный сайт-визитка для фотостудии **ElenaLens (Елена Ленс)**.
Создан на Next.js 16, TypeScript, Tailwind CSS 4, shadcn/ui и Framer Motion.
Деплоится на GitHub Pages через GitHub Actions (полностью бесплатно).

## ✨ Возможности

- **Hero-экран** с именем, статистикой и анимацией
- **Обо мне** — биография, цитата, список техники
- **Услуги** — 6 карточек с ценами и длительностью
- **Портфолио** — галерея с фильтрами по категориям (masonry-раскладка)
- **Процесс** — 4 шага работы
- **Отзывы** — 6 отзывов клиентов с рейтингом
- **Форма заявки** с валидацией и toast-уведомлением
- **Адаптив** под телефон и десктоп
- **Тёмная премиум-палитра** с золотым акцентом
- **SEO**: мета-теги, OpenGraph, семантическая разметка

## 🚀 Локальный запуск

```bash
# установить зависимости
bun install

# запустить dev-сервер
bun run dev
# открыть http://localhost:3000

# собрать статическую версию в out/
bun run build
```

## 📦 Деплой на GitHub Pages

Деплой происходит **автоматически** при пуше в ветку `main` через
GitHub Actions (`.github/workflows/deploy.yml`).

После первого пуша нужно один раз включить Pages:

1. Откройте репозиторий на GitHub → **Settings** → **Pages**
2. В разделе **Build and deployment → Source** выберите **GitHub Actions**
3. Готово. Через 2–3 минуты сайт будет доступен по адресу:
   `https://<ваш-username>.github.io/<имя-репозитория>/`

| Что поменять            | Где                                          |
| ----------------------- | -------------------------------------------- |
| Тексты, цены, услуги    | `src/components/site/services.tsx`           |
| Фотографии в портфолио  | `src/components/site/portfolio.tsx`          |
| Отзывы                  | `src/components/site/testimonials.tsx`       |
| Контакты (телефон, email, соцсети) | `src/components/site/contact.tsx` и `footer.tsx` |
| Цветовая палитра        | `src/app/globals.css` (переменные в `:root`) |
| Шрифты                  | `src/app/layout.tsx` и `globals.css`         |
| SEO-метатеги            | `src/app/layout.tsx`                         |
